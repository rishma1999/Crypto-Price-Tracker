# Project Setup Guide

## Prerequisites

- Node.js (v14 or higher)
- npm (or yarn)

## Running the Web App

1. Navigate to the `web-app` directory:
   ```bash
   cd web-app
2. Install dependencies:
   ```bash
   npm install

   OR

   npm install --legacy-peer-deps
3. Run the development server:
    ```bash
    npm run dev
4. Open your browser and visit http://localhost:3000
